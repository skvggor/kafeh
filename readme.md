# kafeh

---

## Install & run

Install:

```bash
sudo npm i -g kafeh && kafeh
# café!!!
```

Or run with:

```bash
npx kafeh
```

## Parameter (yes | no)

```bash
npx kafeh yes
# café!!!

npx kafeh no
# GET OUT OF HERE!
```
